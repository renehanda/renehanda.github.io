Portifólio

Os códigos que desenvolvi a trabalho pertencem a intranet, então tirei um print das telas ao invés do link.

Página de orçamentos

Página desenvolvida para gerenciar os orçamentos dos contratos de prestação de serviço e materiais, desenvolvida utilizando o Yii framework.

Página de registro de atividades

Página desenvolvida para cadastrar e monitorar as atividades realizadas em home-office, desenvolvida utilizando o Yii framework.

Página de geração de identificadores

Página desenvolvida para geração de identificadores utilizadas para postagem de documentos dos Correios, utilizando PHP e HTML/CSS.

Interface para envio de arquivos via FTP

Aplicação desenvolvida em REXX para selecionar o destino de envio do arquivo via FTP, após selecionar o destino, é solicitado informar o nome do arquivo a ser enviado e então a aplicação monta um JCL para envio do FTP do arquivo a partir do mainframe.
